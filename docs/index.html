Location-based anonymous letter exchange app.

Privacy Policy / プライバシーポリシー

This app collects location data only while in use to enable the core feature of placing and finding letters at real-world locations. Location data is not stored on our servers beyond what is necessary for letter placement.

本アプリは、手紙の設置・発見機能のために使用中のみ位置情報を取得します。位置情報は手紙の設置に必要な範囲でのみサーバーに保存されます。

Data Collection / データ収集

  • Anonymous authentication (no personal information required) / 匿名認証（個人情報不要）
  • Location data (used only for letter placement and proximity detection) / 位置情報（手紙の設置と近接検知のみに使用）
  • Letter text content (stored anonymously) / 手紙のテキスト（匿名で保存）
  • AdMob advertising SDK may collect device identifiers / AdMob広告SDKがデバイス識別子を収集する場合があります

Third-Party Services / 第三者サービス

  • Firebase (Authentication, Firestore) - Privacy Policy
  • Google AdMob - Privacy Policy
